About This Project

This project is made for confused students who wants to clear their doubts on graphs. These tools help in plotting graphs and making graphs with just a equation, it can take multiple types of equations.

Features

linear_scale

Linear Regression

You enter your points in pairs (x, y), and it will produces a graph where all the points will be plotted with a line of best fit drawn over it.

show_chart

Function Grapher
(y=fx) and (y = mx + c)

Plots explicit 2D functions like y = sin(x) or y = x² and y = mx + c. Supports plotting multiple functions, calculating derivatives, and setting custom ranges.

polyline

Implicit Grapher (f(x, y) = 0)

Plots implicit 2D equations where x and y are mixed, such as x² + y² = 1. Uses a contour plot to find where the function equals zero.
